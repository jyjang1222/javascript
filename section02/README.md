# 학습일지
## 학습내용
### 변수의 이름 규칙
1. 맨처음에 숫자와 **$** 와 **_** 를 제외한 특수문자를 사용해서는 안된다.
2. 소문자로 시작하고 대문자로 구분하는 카멜표기법을 사용한다.
3. **공백** 이 있어서는 안된다.
### 변수의 종류
* let
  - let userName = 'Minsu';
* const
  - const pi = 3.14
const를 자주 사용하자
### 연산자의 종류
* +더하기
* -빼기
* *곱하기
* /나누기
* % 나머지
* ** 제곱
* = 할당
### 자료형
* Numbers
  - 2 35 3.14
* Strings
  - '' "" ``

### 기타 조언
1. 끝에 ;을 사용하는 것은 선택이지만 다른 프래그래밍 언어에서는 끝에 ;이 필수인 언어가 많으므로 끝에 ;을 사용하는 것이 좋다.
